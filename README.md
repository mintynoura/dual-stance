Dual Stance
</center>

---

This mod, Dual Stance, introduces the pair up system, inspired by Fire Emblem Awakening+Fates, along with 'crests' from FE: Three Houses.

With a Heart Seal, players can pair up with each other, providing bonuses to support each other as long as they stay close together. Crests can be attached to unpaired Heart Seals akin to a minecraft bundle to provide extra bonuses.
